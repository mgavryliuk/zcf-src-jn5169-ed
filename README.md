# Getting started
1. Register on [NXP](https://www.nxp.com)
2. Go to [NXP Account](https://www.nxp.com/mynxp/home) -> Apps and Services -> Software Licensing and Support -> [View accounts](https://nxp.flexnetoperations.com/control/frse/product?child_plneID=792987&cert_num=173220138)
3. Download [toolchain](https://github.com/openlumi/BA2-toolchain/releases/tag/20201219) and unpack it into `toolchain` directory

# Build
Build is done with CMake
